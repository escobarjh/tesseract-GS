TESSERACT – Rede de Talentos Multidimensionais

Esse projeto foi desenvolvido para a Global Solution do 2º semestre, nas disciplinas de Front-End Design e Web Development.
A ideia é criar uma plataforma que mostra perfis profissionais de um jeito mais voltado ao futuro do trabalho, destacando habilidades técnicas, experiências, interesses e possíveis trajetórias emergentes.
Usei React + Tailwind pra montar uma SPA leve e responsiva, com filtros, modal e páginas extras como login e cadastro.

Usuários e senhas

Instalação do projeto (passo a passo)

Clone o repositório:

git clone https://github.com/escobarjh/tesseract-GS.git


Entre na pasta:

cd tesseract-GS


Instale as dependências:

npm install


Rode o projeto:

npm run dev


Depois disso, a aplicação abre no navegador (geralmente http://localhost:5173).
